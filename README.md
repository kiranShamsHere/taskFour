## 📚 Description
In this repository, I practiced using anchor tags to link different directories, from local to high-level directories, to improve my understanding of HTML navigation.

## 🎯 Features
* Demonstrates the use of anchor tags for navigation
* Links to different directories, including local and high-level directories
* Includes multiple HTML files to showcase navigation between pages

## 📊 Tech Stack
* HTML

## 🗂️ Project Structure
* README.md: This file, containing project information and instructions
* download.jpeg: An image file
* file1.html: An HTML file demonstrating anchor tag usage
* included.html: An HTML file included in other pages
* index.html: The main HTML file
* internalPage.html: An HTML file linked from other pages

## 💻 Installation
No installation required, as this is an HTML project. Simply open the HTML files in a web browser to view.

## 📱 Usage
1. Open the `index.html` file in a web browser
2. Navigate to different pages using the anchor tags
3. View the source code of each HTML file to see the implementation of anchor tags
